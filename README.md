##Requirement

node version: 16.1.0

npm version: 7.13.0

## Getting Started

First, run the development server:

```bash
npm start
# or
yarn dev
```

Second, run the production server:
```bash
npm install
npm run build
# or
yarn
yarn build
```

Open [http://localhost:9000](http://localhost:9000) with your browser to see the result.
